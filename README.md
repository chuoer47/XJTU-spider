# XJTU-spider

本仓库为简单的爬虫项目，爬取的核心数据为西安交通大学教务处官网的相关通知及文件，
可以在`dataset`文件夹中查看爬取到的数据

## 项目启动
安装依赖
```shell
pip install -r requirements.txt
```
启动项目
```shell
python main.py
```

main.py为示例程序，爬取到的数据可以在`dataset/temp`中查看

如果需要制定参数，请在config.py里面修改：
- 爬虫网页
- 改变爬虫的代理
- 改变存储路径
